# UniSwapDiscordBot


>An experimental integration  of UniSwap API  in Discord.Js
Integrate @uniswap/v3-sdk , @uniswap/sdk-core Pool, Trade, Route, Position, NonfungiblePositionManager, nearestUsableTick, with Discord.JS
